Проект симуляции матчей NBA.
nba-sim-core Модуль с логикой CRUD
nba-sim-model Модуль с моделями данных DTO.
Предполагается микросервисная архитектура:
1. микросервис nba_sim, где осуществляется взаимодействия с доменной моделью. Построен на основе RestAPI.
2. микросервис nba_sim_engine, где будет осуществляться непосредственно симуляция матчей (генерация результатов).
Результаты матчей из данного микросервиса будут попадать в очереь сообщений (Apache Kafka).
